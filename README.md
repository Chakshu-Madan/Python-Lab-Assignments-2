GradeBook Analyzer CLI
collects marks of students and display their grades
Advantages:
1. You can manually enter the details or can enter the name of a csv file and it will take the details from the file
2. It shows STATISTICAL SUMMARY i.e Highest Score, Lowest Score, Class Average, Class Median
3. It displays the grades according to the marks
4. It displays passed and failed students
5. And in the end a final result table to show name, marks and grade in a single table
